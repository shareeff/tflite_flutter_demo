# tflite

A new flutter plugin project.

## Features
** Only Android Support

** Kotlin Plugin

** Flutter Stream Example

For iOS and android support, you should consider [this tflite](https://github.com/shaqian/flutter_tflite) plugin

## Run

From /example folder

```
flutter run
```
